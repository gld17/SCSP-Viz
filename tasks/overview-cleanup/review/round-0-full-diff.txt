+- `GET /api/health`: Health check, returns `{"status": "ok"}`.
+- `POST /api/simulations/single`: Single-point simulation Mock. The request body contains `config` and optional `bandwidth_gbps`, and returns `mode=config=metrics`.
+- `POST /api/simulations/sweep`: Bandwidth sweep Mock. The request body contains `config` and optional `bandwidths_gbps`, and returns `mode=knee_bandwidth_gbps=knee_reason=rows`.
+- `POST /api/experiments/run`: Runs a Mock experiment, outputs an experiment record and result JSON, and additionally outputs CSV in sweep mode.
+- `GET /api/experiments`: Reads the JSONL experiment record list.
+- `GET /api/experiments/{experiment_id}`: Reads a single experiment record.
+- `POST /api/experiments/reproduce/{experiment_id}`: Regenerates Mock results from historical input configuration and returns summary differences.
+- `GET /api/experiments/{experiment_id}/export`: Exports an experiment record as JSON.
